Movie API

This web application provides users with access to information about different movies, directors, and genres.
The Movie API retrieves and enables access to this data so that users can create a profile and save data about their favorite movies.

Allow new users to register /users POST A JSON object holding space for user's name and favorite movies. Example: { "name":" ", "favoriteMovies":[] } A JSON object holding user's name, favorite movies, and auto-generated id. Example: { "name": "Mary", "favoriteMovies": [], "id": "ae7c77d6-ae01-498b-9603-11bf345192ce" }
Allow users to update their user information /users/[id] PUT A JSON object holding space for user to update name and/or favorite movie. Example: { "name":"Susan", "favoriteMovies":[] } A JSON object containing a user's id and updated information or a text-based message denying that information was updated. Example: { "id": 2, "name": "Susan Updated Name", "favoriteMovies": [ "Bridesmaids" ] }
Allow new users to add a movie to their list of favorite movies /users/[id]/[movieTitle] POST A JSON object holding space for user's name and new favorite movie/movies. Example: { "name":"Jack", "favoriteMovies":[ "Superbad" ] } A text-based message stating that the movie has been added to the user's array. Example: Superbad has been added to user 1's array.
Allow existing users to remove a movie to their list of favorite movies /users/[id]/[movieTitle] DELETE A JSON object holding space for user's name and new favorite movie/movies. Example: { "name":"Susan", "favoriteMovies":[ "Bridesmaids" ] } A text-based message stating that the movie has been deleted from the user's array. Example: Bridesmaids has been removed from user 2's array.
Allow existing users to unregister /users/[id] DELETE A JSON object holding space for user's id number. Example: { "id": 2 } A text-based message stating that the user has been deleted. Example: User 2 has been deleted.
